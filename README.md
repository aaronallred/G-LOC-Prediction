# G-LOC-Prediction
This repo serves as the primary location for the CU Boulder team's work on modeling G-LOC, including machine learning and deep learning methods. 

The functions in this repo are run to complete either sequential optimization (methods selection for imputation, sliding window parameters, feature reduction, or class imbalance method) or cross validation (performance assessment).

The preliminary development branch serves to assess machine learning methods.

'OldCode' contains the work completed by Vicki Herd. This code is in Matlab and serves as preliminary visualization.

'scripts' contains the most up to date functions for assessing machine learning methods through cross validation, or selecting optimal methods via sequential optimization.

gloc.sh is an example batch file used to run the scripts on the Alpine supercomputer.
